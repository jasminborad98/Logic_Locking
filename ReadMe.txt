Information about Oracle:-
------------------------
==> "a.out" is a executable file, which acts like Oracle Here.
==> Oracle takes 2 input (nb,n).
==>** Value of nb must be equal to 4(nb=4), Otherwise Oracle does not give any output.
==> Oracle Gives us 16 Output.

To Run the Oracle, you have to execute [./a.out 4 5] here 4 is value of nb and 5 is the value of n.


Information about Obfuscated File:-
